# Статичный макет avtomatika.ru

Файлы:
- index.html
- styles.css

Как использовать:
1. Залить оба файла на бесплатный хостинг: Netlify, Vercel, GitHub Pages и т.п.
2. Открыть опубликованный URL.
3. В Figma использовать плагин HTML to Design / html.to.design.
4. Импортировать URL страницы.
5. После импорта заменить заглушки-иллюстрации на реальные изображения/компоненты из текущего Figma-макета.
